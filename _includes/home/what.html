What is The Track?

The Track is a time tracking app for freelancers. It’s easy to use the app, and it comes on multiple platforms (macOS, Windows, Ubuntu)

You can create a project easily, and start tracking time right away, WITHOUT REGISTER. You can name each session, or just save it as “general session”.

Upcoming features are: Auto-stop tracking when freelancer doesn’t work; set reminders for each “X” minutes to make sure you are still working, need a pause, or something else; Use Pomodoro technique while working; calculate your fee for project budget; ONE ACCOUNT, ALL PLATFORM!
